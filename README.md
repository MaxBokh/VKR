# VKR
Дипломный проект НИУ ВШЭ

Тема: Разработка унифицированной методики выбора статистических критериев для анализа продуктовых метрик в условиях интеграции UGC- и SVOD-платформ (на примере слияния Rutube и Premier)

Студент: Бохонко Максим Дмитриевич

Научный руководитель: Карпов Анатолий




СТРУКТУРА РАБОТЫ:

1. Введение

Актуальность: методологический вызов унификации аналитики при слиянии компаний с разными бизнес-моделями.

Цель и задачи.

Объект и предмет исследования.

Научная и практическая новизна.

2. Специфика продуктовых метрик и статистических вызовов в гибридной экосистеме (UGC + SVOD)

2.1. Ключевые метрики и особенности платформы Rutube (UGC-ядро).

2.2. Ключевые метрики и особенности платформы Premier (SVOD-ядро).

2.3. Потенциальные гибридные метрики и гипотезы после интеграции.

2.4. Риски методологического разнобоя в аналитике.

3. Сравнительный анализ и синтез статистических критериев для гибридной платформы

3.1. Критический разбор типовых подходов каждой платформы (на примере сравнения долей и средних).

3.2. Сравнительный анализ критериев на синтетических данных: выявление границ применимости.

3.3. Разработка унифицированной схемы принятия решений (Unified Decision Tree) для выбора критериев.

4. Валидация методики на практических задачах интеграции

4.1. Кейс 1: Гармонизация подходов к A/B-тестированию интерфейсных изменений.

4.2. Кейс 2: Оценка кросс-платформенного эффекта (влияние контента Premier на метрики Rutube и наоборот).

4.3. Разработка практического инструментария (скрипт/дашборд-рекомендатель).

5. Заключение

5.1. Основные выводы.

5.2. Практические рекомендации и дорожная карта внедрения методики в объединённой команде.

5.3. Направления дальнейших исследований.
